# Mortgage Advisor Website

This is a modern, responsive website tailored for a Polish mortgage advisor, built with React and Tailwind CSS. The design is inspired by 2024 UI/UX trends featuring clear visual hierarchy, subtle glassmorphism, and a high-trust, premium aesthetic.

## Tech Stack
* **Vite** (Build Tool)
* **React 18** (UI Framework)
* **Tailwind CSS v3** (Utility-first CSS Framework)

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Krzysztof-Bednarczyk/mortgage-advisor-website.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd mortgage-advisor-website
   ```
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Start the development server:**
   ```bash
   npm run dev
   ```
5. **View the website:** Open your browser and navigate to `http://localhost:5173/`

## Project Structure
* `src/App.jsx` - Main application wrapper and page layout
* `src/components/Hero.jsx` - The top section including the value proposition and contact form
* `src/components/Services.jsx` - The breakdown of financial services offered

## Features
* Mobile-first responsive design
* Interactive form elements with focus states
* Custom gradient backgrounds and shadow effects
* CSS-based entrance animations
